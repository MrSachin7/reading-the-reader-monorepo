# Reading The Reader Frontend

This folder contains the Next.js frontend for Reading The Reader.

## Getting Started

Run the development server from this folder:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) to view the app.

## Common Commands

```bash
bun dev
bun build
bun lint
```
