# Reading The Reader Backend

This folder contains the .NET backend solution for Reading The Reader.

## Common Commands

```bash
dotnet build reading-the-reader-backend.sln
dotnet test reading-the-reader-backend.sln
dotnet run --project src/ReadingTheReader.WebApi/ReadingTheReader.WebApi.csproj
```
