# Endpoints

This section documents backend API endpoints with concrete request and response contract shapes.

## Available Endpoint Docs

- [/endpoints/eyetrackers/](/endpoints/eyetrackers/): discover, select, start, and stop eyetracker tracking.
- [/endpoints/participant/](/endpoints/participant/): set the current participant metadata used by the active experiment session.
- [/endpoints/reader-shell-settings/](/endpoints/reader-shell-settings/): load and update reader/researcher/replay shell view settings.
- [/endpoints/reading-material-setups/](/endpoints/reading-material-setups/): create, list, fetch, and update reusable markdown reading material presets.
